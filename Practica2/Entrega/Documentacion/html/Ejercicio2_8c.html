int main ( void  )

El proceso padre crea cuatro procesos hijos, que imprimen "Soy el proceso hijo <PID>", duermen treinta segundos, y luego imprimen "Soy el proceso hijo <PID> y ya me toca terminar.". Una vez el proceso padre crea cada uno de los hijos, espera 30 segundos y mata al hijo enviando la senal SIGTERM. De esta forma, es imposible que los hijos impriman el segundo mensaje, pues el padre los habra matado antes.
